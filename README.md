# FGH Calculator

## How to use
First, in the input field write any number or use the symbol ω (this input field is after the letter f). Then in the argument in parentheses enter any number, you can use the letter 'e' as an exponent.
Then, click the "Calculate" button.

## Warning
You cannot use letters from different alphabets (including Latin and Cyrillic), you cannot use various symbols and signs. ONLY digits, the symbols '+', '*', '^' and the Greek letter 'ω' are allowed. (+, *, ^ are only allowed when using ω, always placed after ω).

## Alpha
- Numbers 0 - 99 (Everything beyond 99 that takes a long time for the ExpantaNum.js library to compute).
- Omega: ω, ω+k, ωk, ωk+m, ω^2, ω^3, ω^k, ω^ω.

## Maximum
f_ω^ω(n).
